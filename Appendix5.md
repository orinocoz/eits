Appendix 5: NIST CSF 2.0 Category Catalogue

This catalogue lists the CSF 2.0 Functions and Categories defined by NIST in CSWP 29.

| Function code | Function | Purpose |
|---|---|---|
| GV | Govern | Establish and oversee cybersecurity risk strategy, expectations, and policy. |
| ID | Identify | Understand the organization's current assets, context, and cybersecurity risks. |
| PR | Protect | Apply safeguards to reduce the likelihood and impact of cybersecurity events. |
| DE | Detect | Find and analyze possible attacks, anomalies, and compromises. |
| RS | Respond | Take action to contain and manage detected cybersecurity incidents. |
| RC | Recover | Restore affected assets and operations after a cybersecurity incident. |
